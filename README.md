Hotel Booking System
Project Overview
The Hotel Booking System is a web-based application designed to manage hotel room reservations efficiently. It allows users to check available rooms, make bookings, and complete reservations through an online payment system. The system helps hotel management automate the booking process and maintain records easily.

Features
View available hotel rooms

Book rooms online

Manage customer information

Online payment system

Track reservation details

Simple and user-friendly interface

Technologies Used
C#

ASP.NET

HTML

CSS

Bootstrap

Visual Studio

System Modules
Room Management – Manage room availability and room details.

Customer Management – Store and manage customer information.

Booking Management – Handle reservations and booking records.

Payment System – Secure online payment for room booking.
